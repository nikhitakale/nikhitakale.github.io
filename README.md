# Portfolio Website

My personal portfolio showcasing AI/ML testing, blockchain, and web development projects.

## Live Site
https://nikhitakale.github.io

## Quick Updates

### Update Your Bio
Edit the "About Me" section in `index.html` (line ~45):
```html
<p class="about-text">
    Your catchy bio here (2-3 sentences)
</p>
```

### Add Projects
1. Find the TODO comments in `index.html` (around lines 115-165)
2. Update project titles, descriptions, and GitHub links
3. Add your blockchain and Drupal project details

### Deploy Changes
```bash
git add .
git commit -m "Update portfolio"
git push
```

Changes go live in 1-2 minutes at https://nikhitakale.github.io
